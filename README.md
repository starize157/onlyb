# Benz — Automotive UGC Portfolio (Static)

Pure HTML / CSS / JS. No build step. Just open `index.html` or push to GitHub Pages / Netlify / Vercel.

## Files
- `index.html` — page markup
- `styles.css` — design system + layout
- `script.js` — renders the work grid, services list, and footer year
- `assets/` — hero + portfolio images

## Deploy
- **GitHub Pages**: push to `main`, enable Pages in repo settings, root `/`
- **Netlify / Vercel**: drag-and-drop the folder

## Customize
- Copy in `index.html` / `script.js`
- Colors in `styles.css` at the top (`:root { ... }`) — the `--accent` variable is the gold
- Replace images in `assets/` keeping the same filenames, or update the paths in `script.js`
